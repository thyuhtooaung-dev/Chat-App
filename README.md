# Chat-App

A modern real-time chat application built with TypeScript.

## Overview

Chat-App is a contemporary messaging platform that enables users to communicate in real-time. The application is primarily built with TypeScript (97.5%), with supplementary JavaScript and CSS for a polished user experience.

## Tech Stack

- **TypeScript** - Primary language (97.5%)
- **JavaScript** - Supporting scripting (1.9%)
- **CSS** - Styling (0.6%)

## Features

- Real-time messaging
- Modern, responsive UI
- TypeScript for type safety
